## Cluster
- 다른 서버들을 하나로 묶어 하나의 시스템 같이 동작하게 함.
- 시스템에 장애가 발생하면, 서비스가 다른 정상적인 서버에서 동작하며 클라이언트에게 지속적이고 끊임없이 고가용성의 서비스를 제공할 수 있다.
